Work samples

Unus - full stack site

This is a website for a fake company called Unus, who sells a novelty video game console of the same name. The controllers only have one button, and games for the console are designed around it. I am particularly proud of this site's nav bar, which resemble buttons; the current page's button looks pressed. The site uses Bootstrap for its responsive elements. All of the pixel art for Unus was made with Piskell, an online pixel art app.

Dots and Boxes - ReactJS app

This is a recreation of Dots and Boxes made with the ReactJS framework. The lines are all buttons, and clicking them updates the state of the game within the JSX. The above link will take you to the live site, and the source code is on my GitHub page.

Seminar scheduling site - ASP.Net Core MVC

This is a web application for CodeCamp, a theoretical event where people can hear lectures about programming. Users can view the time slots for the lectures and plan their itineraries. A database holds the lecture information, and users log in to save their schedules.
