Work samples

Unus - full stack site

This is a website for a fake company called Unus, who sells a novelty video game console of the same name. The controllers only have one button, and games for the console are designed around it. The project requirements did not include responsiveness, so this is best viewed on a computer; for a mobile-friendly version, see the Bootstrap refactor. I am particularly proud of this site's nav bar, which resemble buttons; the current page's button looks pressed.

Unus - refactored with Bootstrap

This is an updated version of the Unus website. It uses Bootstrap to handle the layout, and it is responsive to screen size. I elected to keep the nav bar structurally the same at all screen sizes because of the charm it adds to the site. All of the pixel art for Unus was made with Piskell, an online pixel art app.

Dots and Boxes - React app

This is a recreation of Dots and Boxes made with the ReactJS framework. The lines are all buttons, and clicking them updates the state of the game within the JSX. The above link will take you to the StackBlitz project since GitHub refuses to serve the JSX files properly.
